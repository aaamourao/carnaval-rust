# carnaval-rust

## Intro

`carnaval-rust` is a library for implementing Deep Learning models using Rust.
It is in its early days, and by now, only `Dense` (Fully Connected) layers are
implemented.

## Features

### Activation Functions

* Sigmoid
* Relu
* Leaky Relu
* Tanh
* Softmax
* *None*

### Layers

* Dense
* Conv2D (in progress)
* MaxPool (soon)
* Flatten (soon)
* Dropout (soon)

### Model Types

* Sequential
